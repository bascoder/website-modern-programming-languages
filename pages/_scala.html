Scala

Scala is een programeer taal die leeft in de zelfde wereld als Java. Sacala aplicaties draaien in de JVM en kunnen met weinig moeite gebruik maken van librarys die geschreven zijn in Java en Java programmas kunnen even makkelijk gebruik maken van Scala librarys. Maar niet aleen is scala een taal die makelijk combineerd met bestaande Java code, ook bied scala de mogelijkheid tot functioneel programeren.

Geschiedenins en adoptie

De eerste publieke versie van scala zag het licht in 2003 en heeft wortels in zowel Pizza als Funnel. Pizza was een functionele taal die compileerde tot Java bytecode en dus kon draaien in de JVM en Funnel was een mimimalistische taal die probreerde functioneel en object georienteerd programeren te mixen. Met Funnel was een van de problemen dat er te weinig librarys waren, deels om dit probleem te ondervangen is er bij Scala voor gekozen om Java compatibiliteid in te bouwen.

Momenteel word Scala door ongeveer 0.002% van de voor w3techs bekende servers gebruikt. Dit getal kan de indruk geven dat Scala er niet toe doet echter niks is minder waar. Scala vooral gebruikt voor websites die grote bezoekers aantallen moeten verwerken. Een paar voorbeelden van Scala gebruikers: Twitter, Tumbler, LinkedIn, Novell. Het mag duidelijk zijn dat al bedient Scala maar een klein aantal bedrijven, is ze toch stiekem wel een grote speler.
