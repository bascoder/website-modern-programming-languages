Scala

Scala is een programmeer taal die leeft in dezelfde wereld als Java. Scala applicaties draaien in de JVM en kunnen met weinig moeite gebruik maken van library's die geschreven zijn in Java en Java programma's kunnen even makkelijk gebruik maken van Scala library's. Maar niet aleen is Scala een taal die makkelijk combineert met bestaande Java code, ook biedt Scala de mogelijkheid tot functioneel programmeren.
