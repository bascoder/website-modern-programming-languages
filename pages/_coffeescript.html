Wat is CoffeeScript

CoffeeScript is een scripttaal die compiled naar JavaScript. CoffeeScript probeert de goede delen van JavaScript naar voren te brengen, omdat de makers geloven dat javaScript slechte delen bevat.
CoffeeScript is compatible met alle JavaScript libraries en frameworks die beschikbaar zijn, andersom geldt hetzelfde omdat CoffeeScript naar JavaScript compiled.

Hoe te gebruiken

Om CoffeeScript te kunnen gebruiken is de compiler benodigd. Het npm pakket coffee-script bevat de compiler. Deze is als volgt te installeren: npm install -g coffee-script Vervolgens kan de code als volgt compiled worden: coffee -C file.coffee #compile to file.js once coffee -w file.coffee #compile and watch for file changes Let er op dat CoffeeScript code altijd in .coffee bestanden staan. Zie de website van CoffeeScript voor meer compiler opties.

Functionaliteit

Functions

Functions worden gedefinieerd als lambda expressies. Bijvoorbeeld: add = (x,y) -> x + y Het bovenstaande code voorbeeld declareert een functie add wat twee parameters neemt, ze optelt en de uitkomst retourneert. Dit compiled naar de volgende JavaScript: // Generated by CoffeeScript 1.10.0 (function() { var add; add = function(x, y) { return x + y; }; }).call(this); De buitenste function is een closure om te voorkomen dat symbolen in de global namespace terecht komen. Vervolgens wordt de functie add gedefinieerd zoals gespecificeerd in het coffee bestand.

Objects

Objecten die volgens de JavaScript object notatie gedefinieerd worden kunnen zowel met als zonder braces geschreven worden: car = {brand: "Opel", color: "blue"} car2 = brand: "Ferrari" color: "red"

Flow control

CoffeeScript ondersteund bekende control flow keywords zoals if en else, maar ook minder voorkomende zoals unless en if als conditional assignment. Blocks in gebruiken geen braces of haakjes, daarentegen wordt whitespace indentation gebruikt om blocks te onderscheiden. Zie het volgende code voorbeeld: weekend = true if saturday or sunday #weekend is true if saturday or sunday are true if someCondition console.log("someCondition is true") else console.log("someCondition is false")

Splats

Het arguments object van JavaScript kan nagebootst worden met …, deze heet splats. logFirstThree = (first, second, third, others...) -> console.log(first); console.log(second); console.log(third); logFirstThree ["harry", "kees", "frans", "fred", "john"] # Outputs: harry kees frans Zoals in het code voorbeeld is te zien kan een array meegegeven worden naar een functie met spats. De eerste paar elementen zijn named en de rest zit in het others object.

Class-based object oriented programming

Tot JavaScript 6 werden classes niet ondersteund, is class-based object oriented programming wel mogelijk. CoffeeScript heeft ondersteuning voor de keywords class, extends en super. Deze keywords kunnen op dezelfde manier gebruikt worden als in Java.

embedded JavaScript

Met het `(back tick) symbool. Is het mogelijk om JavaScript inline te embedden: funkyFunction = `function(a, b) { // JavaScript code return a + Math.pow(a, b); }`
